## рҹҺҜ User Flows

### BEFORE: Finding Property List

```
User wants to view properties

1. Look at navbar
   "Pages" seems right
   
2. Hover over "Pages"
   See 7 options
   "Properties" is first item
   
3. Click "Properties"
   Navigate to property list
   
вҸұпёҸ Steps: 3 clicks, moderate friction
```

### AFTER: Finding Property List

```
User wants to view properties

1. Look at navbar
   See "Property" tab (obvious!)
   
2. Hover over "Property"
   See 3 options
   Choose "Property List"
   
3. Click "Property List"
   Navigate to property list
   
вҸұпёҸ Steps: 3 clicks, low friction вң…
```
