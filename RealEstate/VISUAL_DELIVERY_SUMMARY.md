## 📈 Conversion Funnel

```
1. DISCOVER
   └─ Lands on Home Page
       └─ Clicks "Pages" → Properties

2. BROWSE
   └─ Views Properties List
       └─ Uses Filters
           └─ Clicks "View Details" or "Add to Cart"

3. COMPARE
   └─ Adds to Cart
       └─ Reviews Saved Properties
           └─ Uses Comparison Tool
               └─ Clicks "Proceed to Checkout"

4. INQUIRE
   └─ Fills Checkout Form
       └─ Selects Viewing Date/Time
           └─ Submits Inquiry

5. CONFIRM
   └─ Receives Confirmation
       └─ Agent Contacts User
           └─ Schedules Viewing
```
